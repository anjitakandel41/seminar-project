Welcome

This seminar on Cloud Computing & DevOps (AWS-focused) was conducted at Pokhara University. It covers fundamentals of cloud, AWS services, DevOps practices, CI/CD pipelines, and real-world deployment.

About the Seminar

Duration: 5 Days (30 Hours)
Sessions: 3 per day (2 hours each)
Focus: AWS, DevOps, CI/CD, Monitoring

Syllabus Overview

Day 1: Cloud Foundations

  • Cloud fundamentals & models
  • AWS introduction & ecosystem
  • IAM, Regions, Availability Zones

Day 2: AWS Core Services

  • EC2 (Compute)
  • S3 (Storage)
  • Databases (RDS, DynamoDB)

Day 3: Networking & DevOps

  • VPC, Subnets, Security
  • DevOps lifecycle

Day 4: Git & CI/CD

  • Git & GitHub basics
  • CI/CD pipelines

Day 5: Monitoring & Project

  • CloudWatch monitoring
  • AWS Lambda, SNS
  • Mini project deployment
